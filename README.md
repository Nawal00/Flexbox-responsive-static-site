## Technologies used

* JavaScript
* SCSS/HTML5
* CSS/Flexbox
* Flickity - plugin
* AOS

## Description

I built a homepage for a website following a [photoshop file] as the design and layout of the website. The site has the following features;

- Fully flex box layout
- Slider with images
- Youtube embedded player
- Editable form fields
- Fully Responsive
